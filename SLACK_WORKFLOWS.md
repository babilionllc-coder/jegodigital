## Workflow 1 — `/onboard <client name>`

**Effect:** Cuts client onboarding from 2 hours → 5 minutes. Spawns a private client channel with starter canvas + DMs you the WhatsApp invite link to send.

**Trigger:**
- Type: **Shortcut from a channel** (or "From a link" for slash-command-style)
- Name: `Onboard new client`
- Description: `Spawn private #client-<name> channel with onboarding canvas`
- Run from: any channel (or restrict to `#daily-ops`)

**Form input:**
| Variable | Label | Type | Required |
|---|---|---|---|
| `client_name` | Client name (e.g. "Flamingo") | Single-line text | Yes |
| `service_pack` | Service pack | Dropdown (Pack Crecimiento, Pack Dominación, Custom, Trojan Free) | Yes |
| `monthly_mxn` | Monthly fee (MXN) | Number | Yes |
| `start_date` | Service start date | Date | Yes |
| `primary_email` | Client primary email | Email | Yes |

**Steps:**

1. **Send a message** to channel: (use a variable to dynamically pick `#client-{{client_name | lowercase | replace_spaces_with_dashes}}` — but Workflow Builder can't create channels itself, so this step actually sends to a fallback channel)
   - Alternative: send to `#daily-ops` saying "New client onboarding triggered: {{client_name}}. Create channel #client-{{client_name | lowercase}} now via the + icon."

2. **Send a DM to you** with this message:
```
🎉 New client kicked off — {{client_name}}

Pack: {{service_pack}}
Monthly: ${{monthly_mxn}} MXN
Start: {{start_date}}
Email: {{primary_email}}

⚡ Next steps (in order):
1. Create channel #client-{{client_name | lowercase}} in Slack
2. Send WhatsApp welcome (template below)
3. Schedule kickoff call via Calendly
4. Run /quote {{client_name}} to generate the cotización
5. Post the verified KPIs canvas in their channel

📱 WhatsApp template (copy-paste):
"Hola {{client_name}}, bienvenidos a JegoDigital. Aquí va el link a su Calendly para que agendemos la llamada de kickoff: https://calendly.com/jegoalexdigital/30min ¡Hablamos pronto!"
```

3. **Save to Firestore** (optional, requires webhook step):
   - URL: `https://us-central1-jegodigital-e02fb.cloudfunctions.net/onboardClient`
   - Method: POST
   - Body: all variables as JSON

**Backend Cloud Function needed:** `onboardClient` (see code in section 5 below)

## Workflow 2 — `/lead <name> <company>`

**Effect:** Add a warm lead to the pipeline list with a 24h follow-up reminder.

**Trigger:**
- Type: **Shortcut from a channel**
- Name: `Add warm lead`
- Description: `Quick capture of a warm lead with auto follow-up reminder`
- Run from: `#leads-hot` only

**Form input:**
| Variable | Label | Type | Required |
|---|---|---|---|
| `lead_name` | Lead name | Single-line text | Yes |
| `company` | Company | Single-line text | Yes |
| `source` | Source | Dropdown (Cold email, Cold call, Sofia WA, Sofia IG, Calendly, Audit form, Referral, Other) | Yes |
| `notes` | Notes (1–2 lines) | Multi-line text | No |

**Steps:**

1. **Send a message** to `#leads-hot`:
```
🔥 NEW WARM LEAD — {{lead_name}} @ {{company}}

Source: {{source}}
Captured: {{workflow_started_at}}
Notes: {{notes}}

Next action: Send Calendly OR draft personalized reply within 24h.
```

2. **Send a delay** of 24 hours

3. **Send a DM to you**:
```
⏰ 24h reminder — {{lead_name}} @ {{company}}

Did this lead move? If yes, update status in #leads-hot.
If no, send a follow-up NOW. Cold leads go cold fast.
```

**Backend:** None required — pure Workflow Builder.

## Workflow 3 — `/quote <client>`

**Effect:** Generate a JegoDigital cotización PDF in 90 seconds and post the link to `#revenue`.

**Trigger:**
- Type: **Slash command** (Workflow Builder Premium feature)
- Slash command: `/quote`
- Description: `Generate cotización PDF`

**Form input (parsed from slash args):**
| Variable | Label | Type | Required |
|---|---|---|---|
| `client_name` | Client/prospect name | Single-line text | Yes |
| `service_pack` | Pack | Dropdown (Crecimiento, Dominación, Custom, Trojan Free) | Yes |

**Steps:**

1. **Send to webhook**:
   - URL: `https://us-central1-jegodigital-e02fb.cloudfunctions.net/generateQuote`
   - Method: POST
   - Headers: `X-Slack-Workflow-Token: <generated by you, set as env in Cloud Function>`
   - Body:
```json
{
  "client_name": "{{client_name}}",
  "service_pack": "{{service_pack}}",
  "triggered_by_user": "{{user_id}}",
  "triggered_at": "{{workflow_started_at}}"
}
```

2. **Wait for webhook response** (Workflow Builder supports this — set timeout 90s)

3. **Send a message** to `#revenue`:
```
📄 Cotización ready — {{client_name}} ({{service_pack}})

PDF: {{webhook_response.pdf_url}}
Generated: {{workflow_started_at}}

Next: Send to client via WhatsApp + Calendly link for the call.
```

**Backend Cloud Function needed:** `generateQuote` (see code in section 5)

## Workflow 4 — `/end-day` close-of-day ritual

**Effect:** Forces a clean shutdown of the day with 3 questions. No more "did I do today's big rock?" guilt.

**Trigger:**
- Type: **Scheduled** (run every weekday 8:00 PM Cancun)
- Or: **Shortcut from a channel** for manual trigger

**Form input:** None at trigger (form opens during workflow)

**Steps:**

1. **Send a DM to you** with form attached:
```
🌙 End-of-day check-in — {{date}}

Take 60 seconds. Tomorrow you'll thank yourself.
```

2. **Open form** with these questions:
   - Q1 (Yes/No): Did you ship today's Big Rock?
   - Q2 (Single-line text): If yes, what proof exists? (Slack link, commit hash, post URL)
   - Q3 (Single-line text): What's tomorrow's Big Rock?

3. **Send a message** to `#daily-ops`:
```
✅ End-of-day — {{date}}

Big Rock done: {{q1_answer}}
Proof: {{q2_answer}}
Tomorrow's rock: {{q3_answer}}

Closed at {{workflow_started_at}}.
```

4. **Save to Firestore** (optional webhook):
   - URL: `https://us-central1-jegodigital-e02fb.cloudfunctions.net/saveEndOfDay`
   - Body: `{ date, big_rock_done, proof, tomorrow_rock }`

**Backend:** Optional — pure Workflow Builder works for the Slack-only version.

## 5. Cloud Function backends

### 5a. `onboardClient` (skeleton)

```javascript
// website/functions/slackWorkflows.js
const functions = require("firebase-functions");
const admin = require("firebase-admin");
const axios = require("axios");
const { slackPost } = require('./slackPost');

if (!admin.apps.length) admin.initializeApp();

exports.onboardClient = functions.https.onRequest(async (req, res) => {
    const { client_name, service_pack, monthly_mxn, start_date, primary_email } = req.body;

    // 1. Save to Firestore
    const docRef = await admin.firestore().collection("clients").add({
        client_name,
        service_pack,
        monthly_mxn: Number(monthly_mxn),
        start_date,
        primary_email,
        status: "onboarding",
        created_at: admin.firestore.FieldValue.serverTimestamp(),
    });

    // 2. Trigger client-onboarding skill via webhook OR enqueue a task
    // (full skill implementation in skills/client-onboarding)

    // 3. Post confirmation to #revenue
    await slackPost('revenue', {
        text: `🆕 New client onboarded — ${client_name}`,
        blocks: [
            { type: "header", text: { type: "plain_text", text: `🆕 ${client_name} onboarded` } },
            { type: "section", text: { type: "mrkdwn", text: `Pack: *${service_pack}*\nMRR: $${monthly_mxn} MXN\nStart: ${start_date}` } },
        ],
    });

    res.status(200).json({ ok: true, client_id: docRef.id });
});
```

### 5b. `generateQuote` (skeleton)

```javascript
exports.generateQuote = functions
    .runWith({ timeoutSeconds: 120, memory: "1GB" })
    .https.onRequest(async (req, res) => {
        const token = req.headers['x-slack-workflow-token'];
        if (token !== process.env.SLACK_WORKFLOW_TOKEN) {
            return res.status(401).json({ ok: false });
        }

        const { client_name, service_pack } = req.body;

        // Trigger jegodigital-cotizaciones skill (HTML+WeasyPrint pipeline)
        // For now, return a placeholder — full implementation in skill
        const pdf_url = `https://jegodigital.com/cotizaciones/${client_name.toLowerCase().replace(/\s/g,'-')}-${Date.now()}.pdf`;

        // TODO: actually generate via skill
        // const result = await invokeSkill('jegodigital-cotizaciones', { client_name, service_pack });

        res.status(200).json({
            ok: true,
            pdf_url,
            client_name,
            service_pack,
        });
    });
```

### 5c. `saveEndOfDay` (skeleton)

```javascript
exports.saveEndOfDay = functions.https.onRequest(async (req, res) => {
    const { date, big_rock_done, proof, tomorrow_rock } = req.body;
    await admin.firestore().collection("end_of_day").doc(date).set({
        big_rock_done,
        proof,
        tomorrow_rock,
        saved_at: admin.firestore.FieldValue.serverTimestamp(),
    });
    res.status(200).json({ ok: true });
});
```
